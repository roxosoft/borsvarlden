Lead .NET Developer
В настоящее время в нашей Компании открыта вакансия .NET / C# Разработчика на один из текущих Проектов, и нас заинтересовала Ваша кандидатура.

Коротко о Проекте:

Проект представляет из себя десктоп приложение для быстрой обработки большого количества файлов различных форматов. Приложение поддерживает большое количество форматов и инструментов их обработки, и предоставляет пользователю интеллектуальный интерфейс для манипуляций с файлами.

В рамках проекта необходимо решать следующие задачи:

• добавление новых инструментов
• поддержка новых форматов
• исследование различных фреймворков и библиотек для выполнения вышеописанных задач
• работа с установщиком

Основные требования:

• C# как основной язык программирования
• .NET Framework
• разговорный английский (upper-intermediate и выше)

Мы предлагаем:

• возможна удаленная работа из дома или в нашем офисе в СПб
• работа с передовыми технологиями в интернациональной команде
• возможность командировок заграницу
• конкурентная зарплата по результатам собеседования

Частично работа выполняется под руководством Менеджера Проекта нашей Компании, в некоторых случаях требуется взаимодействие напрямую с Клиентом.
